# BoboFood – E-commerce Android Application

BoboFood is an Android **e-commerce app** that allows users to browse and order food easily from local restaurants.

## Features
- Browse and search food items
- Add items to cart and place orders
- User authentication with Firebase
- Offline support using Room database
- Smooth UI with Kotlin Coroutines for better performance

## Technologies Used
- **Kotlin** – main programming language
- **MVVM (Clean Architecture)** – project architecture
- **Dependency Injection** – for scalable and maintainable code
- **Firebase** – authentication and backend services
- **Retrofit** – API communication
- **Room** – local database
- **Kotlin Coroutines** – asynchronous operations
